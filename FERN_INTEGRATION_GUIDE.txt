Fern-WiFi-Cracker Integration Guide
=================================

This guide explains how to install and use fern-wifi-cracker with the WiFi Penetration Testing Tool.

Installing Fern-WiFi-Cracker
============================

Method 1: Automatic Installation (Recommended)
--------------------------------------------
1. Run the install_fern_kali.bat script from Windows:
   - Double-click install_fern_kali.bat
   - This will automatically upload and run the setup script on your Kali VM

Method 2: Manual Installation on Kali Linux
------------------------------------------
1. Connect to your Kali VM via SSH:
   ssh vaptrix@172.18.0.1

2. Update package list:
   sudo apt update

3. Install fern-wifi-cracker:
   sudo apt install fern-wifi-cracker

4. Verify installation:
   which fern-wifi-cracker

Using Fern-WiFi-Cracker with the Tool
====================================

Command Line Usage
------------------
1. Run the tool with fern integration:
   sudo python3 wifi_penetration_tool/main.py --use-fern

2. Specify interface if needed:
   sudo python3 wifi_penetration_tool/main.py --use-fern -i wlan1

GUI Usage
---------
1. Run the SSH GUI:
   run_ssh_gui.bat

2. Connect to your Kali VM (172.18.0.1, vaptrix, Xevyte@2025)

3. Use the "Fern Integration" section to:
   - Check if fern is available
   - Install fern if needed
   - Run fern scans

4. Or use the "Run with Fern" button in the WiFi Tools section

Troubleshooting
===============

Common Issues
-------------
1. "fern-wifi-cracker: command not found"
   - Reinstall using the automatic installer
   - Check PATH environment variable

2. "Permission denied" errors
   - Ensure you're running with sudo/root privileges
   - Check file permissions on fern installation

3. SSH connection failures
   - Verify Kali VM is running
   - Check IP address and credentials
   - Ensure SSH service is enabled on Kali

4. GUI not responding
   - Restart the GUI application
   - Check network connectivity
   - Verify fern is properly installed

Security Considerations
=======================
